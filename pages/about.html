HTML: HyperText Markup Language

HTML adalah bahasa markup yang digunakan untuk membuat halaman web. Dengan HTML, Anda dapat membuat teks, gambar, audio, video, dan berbagai jenis konten lainnya yang dapat ditampilkan di web.

Text Formatting

Berikut adalah beberapa tag HTML yang dapat digunakan untuk memformat teks:

  • <strong> untuk membuat teks menjadi tebal
  • <em> untuk membuat teks menjadi miring
  • <u> untuk membuat teks bergaris bawah
  • <s> untuk membuat teks dicoret
